00252 /*
00253  * struct member types:
00254  * private: you may look at them, but not change them,
00255  *          and they might not mean anything to you.
00256  * ro: can be read anytime, but only changed when the watcher isn't active.
00257  * rw: can be read and modified anytime, even when the watcher is active.
00258  *
00259  * some internal details that might be helpful for debugging:
00260  *
00261  * active is either 0, which means the watcher is not active,
00262  *           or the array index of the watcher (periodics, timers)
00263  *           or the array index + 1 (most other watchers)
00264  *           or simply 1 for watchers that aren't in some array.
00265  * pending is either 0, in which case the watcher isn't,
00266  *           or the array index + 1 in the pendings array.
00267  */
